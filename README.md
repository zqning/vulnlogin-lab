## 📖 项目简介

VulnLogin-Lab 是一个专门用于练习**登录界面渗透测试**的脆弱 Web 应用。项目灵感来源于微信公众号文章《[登录界面的20种渗透思路 —— 图文解析](https://mp.weixin.qq.com/s/7kIGI5CyfDr8vBl_3Yfm7A)》，感谢原作者 **泷羽Sec** 和 **智榜样网络安全学习中心** 的分享。

本项目将文章中的 20+ 种攻击思路转化为可实践的漏洞场景，涵盖从经典漏洞到现代攻击手法，适合网络安全学习者、渗透测试初学者进行本地实验。

## 🚀 快速开始

### 环境要求
- Docker Desktop 或 PHP 7.4+ 环境
- Apache/Nginx Web 服务器

### 使用 Docker 部署（推荐）

```bash
# 克隆仓库
git clone https://github.com/zqning/vulnlogin-lab.git
cd vulnlogin-lab

# 启动靶场
docker compose up -d --build

# 访问 http://localhost:8080
